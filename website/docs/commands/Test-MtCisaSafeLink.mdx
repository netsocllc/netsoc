## SYNOPSIS

Checks state of URL block list

## SYNTAX

```powershell
Test-MtCisaSafeLink [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

URL comparison with a block-list SHOULD be enabled.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtCisaSafeLink
```

Returns true if URL block list enabled
