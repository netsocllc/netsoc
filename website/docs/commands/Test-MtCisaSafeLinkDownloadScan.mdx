## SYNOPSIS

Checks state of URL direct download scans

## SYNTAX

```powershell
Test-MtCisaSafeLinkDownloadScan [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Direct download links SHOULD be scanned for malware.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtCisaSafeLinkDownloadScan
```

Returns true if URL direct download scan enabled
