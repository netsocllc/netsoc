## SYNOPSIS

Checks state of baseline CISA rules for DLP in EXO

## SYNTAX

```powershell
Test-MtCisaDlpBaselineRule [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

At a minimum, the DLP solution SHALL restrict sharing credit card numbers, U.S.
Individual Taxpayer Identification Numbers (ITIN), and U.S.
Social Security numbers (SSN) via email.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtCisaDlpBaselineRule
```

Returns true if baseline rules are enforced
