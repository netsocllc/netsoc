## SYNOPSIS

Resets the local cache of Exchange Online queries.
Use this if you need to force a refresh of the cache in the current session.

## SYNTAX

```powershell
Clear-MtExoCache [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

By default all requests are cached and re-used for the duration of the session.

Use this function to clear the cache and force a refresh of the data.

## EXAMPLES

### EXAMPLE 1

```powershell
Clear-MtExoCache
```

This example clears the cache of all EXO requests.
